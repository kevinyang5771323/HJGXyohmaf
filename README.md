# Python图书借阅管理系统

## 核心功能
- 用户登录注册
- 图书借阅与归还
- 图书管理
- 读者管理

## 适用场景
- 学校图书馆
- 企业内部图书室
- 公共图书馆

## 技术特点
- 使用Flask框架进行开发，具备良好的扩展性和稳定性
- 采用MySQL数据库进行数据存储，确保数据安全可靠
- 前端采用HTML、CSS和JavaScript进行设计，界面友好易用
